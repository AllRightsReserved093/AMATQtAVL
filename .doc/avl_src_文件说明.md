# AVL 3.52 `src` 文件详细说明

本文档对应目录：`AVL3.52src/src`。

说明规则：
- `输入` 包括：子程序参数、命令行/交互命令、读取文件、`INCLUDE/COMMON` 全局状态。
- `输出` 包括：返回值/参数回写、全局状态更新、屏幕输出、文件输出、绘图。
- `默认编译` 以 `src/Makefile` 的 `AOBJS + MATRIX` 为准。
- 已排除 Apple 元数据文件 `._*` 和编辑器临时文件 `#*#`。

## 1) 主流程与核心气动

| 文件 | 功能 | 输入 | 输出 | 默认编译 |
|---|---|---|---|---|
| `avl.f` | 主程序入口，命令循环（`.OPER/.MODE/LOAD/MASS/CASE/...`） | 命令行参数（几何/工况/质量文件），用户交互命令，`AVL.INC/AVLPLT.INC` 全局数据 | 调度各模块，更新全局状态，控制台菜单/结果，调用绘图与求解 | 是 |
| `ainput.f` | 读取几何配置文件（机翼/体/控制面/剖面等） | 配置文件文本，输入逻辑单元号，字符串解析工具 | 几何与网格定义写入 `COMMON`，错误标志 | 是 |
| `amake.f` | 根据输入几何生成面元、条带、涡格及关联索引 | 来自 `ainput` 的几何参数与剖面数据 | 离散化后的气动网格与索引、后续求解所需结构 | 是 |
| `asetup.f` | 求解前数据准备（法向、控制变量、引用量等） | 全局几何/运行参数 | 预处理状态（供 AIC/求解器使用） | 是 |
| `aic.f` | 影响系数矩阵与诱导速度相关核心计算（含 `VORVELC` 等） | 网格几何、边界条件、马赫修正参数 | AIC 与诱导速度相关矩阵/中间量 | 是 |
| `aero.f` | 气动载荷与导数主计算（总量、条带、分量等） | AIC/几何/运行状态（迎角、侧滑、角速度、控制量） | 力/矩系数、导数、分布载荷写入全局 | 是 |
| `aoper.f` | `.OPER` 操作点求解与工况管理 | 用户在 OPER 菜单输入、Run case 参数、约束设置 | 解算结果、工况表、调用输出/绘图 | 是 |
| `atrim.f` | 配平与约束映射（变量-约束关系、配平设置） | 配平命令参数、变量/约束索引 | 更新工况约束，配平所需线性系统数据 | 是 |
| `amode.f` | `.MODE` 线化动力学与特征值/特征向量分析 | 已求得工况、质量参数、导数矩阵、用户选择 | 模态特征值/特征向量、模态展示与输出 | 是 |
| `atpforc.f` | Trefftz 面/远场相关力分解与后处理 | 涡系与远场积分所需状态 | 诱导阻力等相关分量 | 是 |
| `amass.f` | 质量文件读入、单位体系、质量/惯量与附加质量处理 | `.mass` 文件、单位定义、几何信息 | 质量属性、惯量、单位派生量，状态标志 | 是 |
| `aoml.f` | 截面/外形相关附加模型（如剖面厚度/修正） | 剖面几何参数、局部流动参数 | 剖面修正量或派生几何/气动量 | 是 |
| `cdcl.f` | `CD(CL)` 极线查值与导数 | 极线参数 `cdclpol`、`CL` | `CD` 与 `dCD/dCL` | 是 |
| `getvm.f` | 输出速度/矩阵类调试或诊断信息 | 全局求解状态、逻辑单元号 | 文本输出到屏幕/文件 | 是 |
| `aoutput.f` | 结果输出总控（总力、分面、分条带、导数等） | 当前求解结果、输出选项 | 格式化输出到终端或文件 | 是 |
| `aoutmrf.f` | MRF（多参考系/旋转框架）相关输出 | 工况与结果数据、文件标识 | MRF 专项结果文本输出 | 是 |

## 3) 数值方法与通用工具

| 文件 | 功能 | 输入 | 输出 | 默认编译 |
|---|---|---|---|---|
| `matrix.f` | 内置 LU 分解/回代（自包含，较慢） | 方阵 `A`、右端项 `B` | LU 因子、解向量 | 是（默认） |
| `matrix-lapackdp.f` | LAPACK 双精度 LU/回代封装（`DGETRF/DGETRS`） | 与 `matrix.f` 同 | 更快求解（依赖 LAPACK） | 否（可切换） |
| `matrix-lapacksp.f` | LAPACK 单精度封装（`SGETRF/SGETRS`） | 与 `matrix.f` 同 | 单精度快速求解 | 否 |
| `matrix-lapacksubs.f` | LAPACK/BLAS 子程序集合（大型依赖包） | 线代例程参数 | 供线代调用 | 否 |
| `matrix-linpack.f` | LINPACK/BLAS 风格实现集合 | 线代例程参数 | 供线代调用 | 否 |
| `spline.f` | 样条插值与导数工具 | `x` 网格、样条系数、查询点 | 插值值/导数 | 是 |
| `sgutil.f` | Akima/几何插值等数值工具 | 离散点列与查询点 | 平滑插值结果 | 是 |
| `autil.f` | 线代与向量通用工具（如 3x3 逆矩阵等） | 小矩阵/向量参数 | 工具函数结果 | 是 |
| `airutil.f` | 剖面数据读取与处理工具 | 翼型文件、点序列 | 归一化/重采样后的翼型数据 | 是 |
| `ba_trans.f` | 坐标/导数变换（机体轴到风轴/稳定轴） | 姿态角、向量/矩阵 | 变换后矩阵/导数 | 否 |
| `vorvel.f` | 涡段诱导速度计算（基础版） | 观测点、涡段端点、边界标志 | 诱导速度向量及导数项 | 否 |
| `vorvelc.f` | 涡段诱导速度计算（改进/压缩修正版） | 同上 | 同上（更稳定或更完整） | 否（功能并入 `aic.f`） |
| `second.f` | 计时接口（由 Makefile 从特定实现复制生成） | 无/系统时钟 | 秒级 CPU 时间 | 是（目标文件） |
| `second_g77.f` | `SECONDS` 的 g77/gfortran 实现 | 系统 `SECNDS` | 秒值 | 间接是（被复制到 `second.f`） |
| `second_ifc.f` | `SECONDS` 的 Intel/其它实现 | `CLOCKX` 微秒计时 | 秒值 | 否（可切换） |
| `dtest.f` | 独立测试程序入口 | 测试输入（程序内/文件） | 测试结果输出 | 否（单独目标） |

## 4) 头文件与内存模块

| 文件 | 功能 | 输入 | 输出 | 默认编译 |
|---|---|---|---|---|
| `AVL.INC` | AVL 全局 `COMMON` 定义（状态、几何、工况、结果） | 被各 `.f` `INCLUDE` | 提供共享变量命名与布局 | 是 |
| `AVLPLT.INC` | 绘图相关全局变量 `COMMON` | 被绘图相关源码 `INCLUDE` | 提供绘图状态共享 | 是 |
| `ADIMEN.INC` | 主维度上限（`NVMAX/NSMAX/...`） | 编译期常量定义 | 决定数组容量与规模上限 | 是 |
| `AINDEX.INC` | 变量/约束/参数索引常量 | 编译期常量定义 | 统一编号语义 | 是 |
| `MASKS.INC` | 绘图掩码、线型等常量 | 被绘图文件 `INCLUDE` | 绘图样式与掩码配置 | 是 |
| `avl_heap_inc.f90` | F90 `module`：可分配 AIC 大数组声明 | `avl_heap.f90` 与调用方 | 统一大数组接口（`AICN/WC_GAM/WV_GAM`） | 否（但源码依赖） |
| `avl_heap.f90` | 分配/释放堆内存数组（`avlheap_init/clean`） | 无（内部调用 module） | 大数组分配状态改变 | 否（Makefile 需手工补） |
